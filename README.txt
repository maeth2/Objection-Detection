Objection Detection Web App based on YOLOV8 Detection by Ultralytics and FastAPI

Install Packages -> pip install -r requirements.txt

TO RUN:
    Navigate to OBJECT DETECTION PROJECT folder
    Open run.bat
    Follow Link to Public URL for localhost:8080

TO RUN MANUALLY(Type in CMD):
    Start Backend Local Host -> uvicorn main:app --reload
    Start Frontend Local Host -> http-server

Note: Make sure to have node.js installed on your computer and set in PATH env

To edit or install additional packages to the backend API:
    Activate Virtual Enviorment -> ./.venv/Scripts/activate
